删除链表中的节点

2021-02-18

[237. 删除链表中的节点]()
'''
删除链表指定节点
Solution：无法获取上一个节点，用下一个节点覆盖
'''

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def deleteNode(self, node):
        node.val = node.next.val
        node.next = node.next.next

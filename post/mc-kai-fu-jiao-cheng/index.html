MC开服教程

2021-02-18

开服教程（Linux）

  1. 先安装java

  2. 去https://yivesmirror.com/downloads/paper下载forge版jar，版本自己选就好。

    便携命令wget http://tcpr.ca/files/mcpc/mcpc-plus-1.12.2-R2.1-forge965-B243.jar

  3. 编写start.sh

    java -Xms256m Xmx1024m -jar 1.14.jar nogui
    java -Xms最小内存 Xmx最大内存 -jar 下载的文件 nogui[是否开启gui]

  4. 试运行bash start.sh生成一系列文件

  5. 修改eula.txt 把false改为true

  6. 配置文件为severs.properties，online-mode为正版验证

补充：

  1. op权限最高为4，op 用户名 即可给予权限

  2. mod放在mods文件夹即可

  3. 备份地图即备份world文件夹即可

  4. 纯净服去下正常jar运行即可

服务器注意事项

  1. 离开终端后停止运行用screen即可解决
  2. 停止运行输入stop，不要直接关闭，否则会丢失数据

python模拟浏览其操作

2021-02-18

起因：蓝皮君大大的镜头提纯太多，不想一个一个去点就写了这个python脚本

前置工作：

  • Anaconda3安装：清华源的Anaconda百度即可
  • chromedriver下载 ：这里提供Chrome版的下载链接：http://chromedriver.storage.googleapis.com/index.html
     各大浏览器可参考：https://docs.seleniumhq.org/download/
    下载好后放到Anaconda安装目录即可
  • 安装seleium
     pip install seleium   ps:推荐将pip换为清华源

脚本编写：

  1. 获取百度云链接与提取码：哔哩哔哩个人投稿页id为"submit-video-list"的div下<ul class="list-list">含有带简介的视频信息。
  2. url清洗：个人是将清洗为 下载链接+提取码 的csv格式
  3. 编写脚本：
 from selenium import webdriver 
 import time
 import os
 f = open("urls.csv", "r")
 lines = f.readlines()
 for line in lines:
     driver = webdriver.Chrome()
     url_list = line.split(",,")
     url = url_list[0]
     captch = url_list[1]
     driver.get(url)
     input_box = driver.find_element_by_id('accessCode')
     input_box.send_keys(captch+"\n")
     time.sleep(3)
     download_button = driver.find_element_by_xpath("//*[@id='layoutMain']/div[1]/div[1]/div/div[2]/div/div/div[2]/a[2]")
     download_button.click()
     time.sleep(3)
     driver.close()
     ```

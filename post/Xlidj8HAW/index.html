Eratosthenes筛法

2021-02-18

描述：素数筛法，可以生成n以下的素数表
英文wiki给出 伪码
中文wiki给出 几重常见代码
百度百科的也给出了几种代码，但个人感觉不太好

算法说明

  1. 生成到n的序列
  2. 将第一个素数p（也就是2）的2倍，3倍，……，k倍做标记也就是筛去
  3. 之后对下一个素数进行同样操作

代码（注：output为从0到n的bool list）

def eratosthenes(n):
    Isprime = [True] * (n+1)
    Isprime[0] = False
    Isprime[1] = False
    for i in range(2,int(n ** 0.5)+1):
        if Isprime[i]:
            for j in range(i*i,n+1,i):
                Isprime[j] = False
    return Isprime

ps:

  1. Q：为什么从i*i开始循环？
      A：i*i 以下的合数已经素数前面筛过
  2. Q：为什么只需只循环到√n？
      A：√n到n之间的合数必然被小于√n的素数筛过（此处可用反证法做证明）
